# AllDayRun
##JSP/Servlet Project

## 프로젝트 소개
러닝 데이터를 기반으로 사용자 경험을 극대화한 웹 서비스로, 운동 애호가들의 러닝 기록 관리와 커뮤니티 참여를 돕기 위해 설계되었으며, 
JSP/Servlet 기반의 웹 구조와 Oracle, MyBatis를 활용한 안정적인 데이터 처리 설계를 통해 러닝 기록과 커뮤니티 기능을 하나의 플랫폼에서 보여주기 위한 프로젝트 입니다.

## 기술 스택
Java · Oracle · JSP · Tomcat · HTML5 · CSS3 · JavaScript · MyBatis

## 실행 방법
1. Eclipse에서 프로젝트 Import  
2. Tomcat 9.0 실행  
3. http://localhost:8080/AllDayRun/index.do 접속
